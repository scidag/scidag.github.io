# 3. 扩展：

1.DIscoveryClient 获取服务实例列表

2.LoadBalancerClient 负载均衡的选择一个实例（要引入 spring-cloud- start - loadbalancer）

3.RestTemplate 可以发起远程调用

4. 引入 spring-cloud-start-alibaba-nacos-config 依赖配置 Nacos 地址

5 添加 数据集（data-id），使用 spring.config.import 导入数据集

6.@Value + @RefreshScope 取值 + 自动刷新

7@ConfigurationProperties 批量绑定自动刷新

8NacosConfigManager 监听配置变化
